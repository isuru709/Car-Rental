# Assets Folder

This folder contains all images and media assets for the DriveX Car Rental landing page.

## Required Images

### Hero Section
- `hero-car.png` - A luxury sports car image with transparent background (recommended size: 1200x600px)

### Fleet Section Cars
- `car-mercedes.png` - Mercedes S-Class image with transparent background (400x250px)
- `car-porsche.png` - Porsche 911 image with transparent background (400x250px)
- `car-bmw.png` - BMW X7 image with transparent background (400x250px)
- `car-tesla.png` - Tesla Model S image with transparent background (400x250px)

## Image Guidelines

1. **Format**: Use PNG with transparent backgrounds for car images
2. **Quality**: High resolution images for best visual impact
3. **Orientation**: Cars should face left or at a 3/4 angle for best presentation
4. **Style**: Prefer studio-style shots with clean lighting
